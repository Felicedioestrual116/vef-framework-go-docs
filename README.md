## 🧭 What this is

vef-framework-go-docs is the official documentation site for VEF Framework Go.

It helps you learn how to use a resource-driven Go backend framework built on FX, Fiber, and Bun. The site is made for people who want clear steps, plain language, and a path that is easy to follow on Windows.

Before you start, make sure you have:

- A Windows PC
- A modern web browser like Chrome, Edge, or Firefox
- An internet connection
- Permission to open downloaded files on your device

If you plan to read the docs in a local browser or work with the framework later, these tools help:

- Git
- Node.js 18 or later
- A code editor like Visual Studio Code

## 🪟 How to run on Windows

Follow these steps on a Windows PC:

1. Open the download or project page in your browser.
2. Download the file or files that match your setup.
3. If the download is a ZIP file, right-click it and choose Extract All.
4. Open the extracted folder.
5. Look for a file named `README.md`, `index.html`, or a setup file if one is provided.
6. If the project includes a local site, open it in your browser.
7. If the project uses Node.js, open PowerShell in the project folder and run:
   - `npm install`
   - `npm start`
8. If the project uses a static preview, open the main HTML file in your browser.
9. Keep the browser open while you read the docs.

If Windows asks whether to trust the file, check that it came from the GitHub link above before you open it.

## 🧭 Troubleshooting

If the page does not open:

- Check that the download finished
- Extract the ZIP file again
- Open the correct file in the folder
- Refresh the browser
- Try a different browser
- Make sure your internet connection is active

If the local site does not start:

- Open PowerShell in the project folder
- Check that Node.js is installed
- Run `npm install` again
- Run `npm start` again
- Look for error text in the terminal window

If a file will not open:

- Right-click it and choose Open
- Check file permissions
- Move the folder to a simple path like `C:\vef-framework-go-docs`
